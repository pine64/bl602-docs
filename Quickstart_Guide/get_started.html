1. Quick Start Guide¶

1.1. Prerequisites¶

  • A BL602 development board such as:

  • PineCone

  • Doi.am DT-BL10, or

  • a first-party BL602 EVB (These were apparently available by way of an early-adopter program.)

  • A PC (running Windows or Linux; macOS may work - please contribute documentation if so!)

1.2. Preparing the environment¶

To get started, follow the instructions for your preferred development host platform:

  • Linux (Ubuntu LTS is used, but any relatively modern version and distribution should work.)

  • Windows using MSYS (Windows 7 or newer is recommended.)

  • Windows with WSL (Windows 10 is required.)

  • macOS (Xcode Tools are required; Homebrew is optional, but recommended.)
